# Yuna System

Purpose:
Maintain a consistent fictional Korean fashion model character named Yuna across multiple image generations.

Core rule:
Preserve identity first. Change angle, pose, outfit, or background only after identity consistency is stable.

Primary anchor priority:
1. Face identity
2. Hair identity
3. Body proportions
4. Outfit silhouette
5. Lighting/style

Best use:
- Load 1–3 strongest reference images
- Use the master identity prompt
- Add only one major variable change per generation
